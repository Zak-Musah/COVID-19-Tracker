## Quick Start

Get up and running with the following.

- Clone from Github

```bash
git clone https://github.com/Zak-Musah/COVID-19-Tracker.git

cd COVID-19-Tracker
npm install
npm start [Automatically launches app in your default browser on http://localhost:3000]

```
